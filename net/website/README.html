Vue 3 + TypeScript + Vite

This template should help get you started developing with Vue 3 and TypeScript in Vite. The template uses Vue 3 <script setup> SFCs, check out the script setup docs to learn more.

Learn more about the recommended Project Setup and IDE Support in the Vue Docs TypeScript Guide.

安装依赖

npm i

调试命令

npm run dev

构建命令

npm run build

构建说明

目录说明

/src
/config 项目主要配置
/language 多语言模块
/layout 项目基本布局
/stores 状态管理
/utils 工具集
/views 页面
App.vue 页面入口
main.ts 脚本入口
package.json 依赖管理
vite.config.ts 构建配置

页面标题

页面logo 放在/public里，对应文件/index.html 网站名字，logo都可在里面进行配置

SEO配置

文件/index.html里面的meta标签，已经声明好了，description是网站说明，keywords是网站关键字，具体内容填写content即可

网页配置

官网主要文案，图标，banner等等配置都在 /config 中可以找到，相关配置 文件中有注释说明

子目录部署

  1. 修改vite.config.ts 下的base 配置为子目录
  2. 同步修改/config/conf_page.ts 下的global.base_url 同步vite.config.ts下的base配置

例： www.baidu.com/rgame 为网站根目录
vite.config.ts文件需配置为 base:’/rgame’
/config/conf_page.ts文件需配置为 base_url:’/rgame’

多环境配置

  1. package.json 新增命令（如：”build:prod01”: “vite build –mode prod01”）
  2. 新增 .env.prod01 文件，增加环境变量 NODE_ENV
  3. 项目中使用 import.meta.env.VITE_USER_NODE_ENV 读取(NODE_ENV 对应 VITE_USER_NODE_ENV)

多语言配置查询

路径：src/language/lang/${语言}.json

各个渠道下载统一方法

路径：src/utils/logicTool.ts
